## How to Use in PowerPoint

1. After deploying to GitHub Pages, your map URL will be:
   `https://YOUR_USERNAME.github.io/us-tariff-map/`

2. In PowerPoint:
   - Go to **Insert ‚Üí Add-ins ‚Üí Get Add-ins**
   - Search for **"Web Viewer"** and install it
   - Add Web Viewer to your slide
   - Paste your GitHub Pages URL

3. The interactive map will be embedded directly in your presentation!
